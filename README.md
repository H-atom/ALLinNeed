# ALLinNeed

### 简介

本系统实现的是一个电影院卖票系统，包含顾客、管理人员以及客服三种角色，主要实现了

1.  定位功能
2.  排片功能
3. 数据查看功能
4. 图形化选座功能
5.  聊天功能
6.  订单管理
7. 通知管理
8.  影片过滤功能

### 环境

Vue2 + springboot2，jdk-15，node.js，使用了element-ui、vuex、vue-router、v-charts，利用BMap的api定位用户位置，后端采用了mybatis-plus和webSocket
